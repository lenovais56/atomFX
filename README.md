## 🔨 Principais táticas de criação do projeto

- `1` `Emmet`: Escrita agil do HTML.
- `2` `CSS`: Desenvolvimento feito com flexbox.
- `3` `Menu responsivo`: Menu responsivo tipo hamburguer desenvolvido para telas de 320px a 768px.
